# iOS-Library-Binary-Practice 

仓库存放源码，`pod update`后自动编译成framework引入工程  

## 效果图  

1. 执行`pod update`后没有做其他操作  
2. `Installing iOSBinaryPractice (1.0.0)`比较慢是因为在编译  
3. Pods文件夹内最终引用`iOSBinaryPractice.framework`
